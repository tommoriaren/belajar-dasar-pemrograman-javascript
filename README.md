Berikut adalah poin-poin utama yang dibahas dalam kelas ini:
* **Dasar JavaScript:** Variabel, tipe data, operator, dan komentar.
* **Struktur Data:** Object, Array, Map, dan Set.
* **Logika Alur Kerja:** If/Else, Switch-Case, For Loop, dan While.
* **Function:** Arrow Function, Parameter, Rest Parameter, dan Closure.
* **Object-Oriented Programming (OOP):** Class, Inheritance, Method, dan Property.
* **Functional Programming:** Pure Function, High-Order Function (Map, Filter, Reduce).
* **Setup Environment:** Node.js, NPM, dan penulisan kode dengan gaya konsisten.
* **Testing:** Melakukan pengujian kode otomatis menggunakan modul bawaan Node.js (`node:test`).
